# Dataset Notes

## Current Default Experiment Dataset

The default command used in this repository runs in synthetic mode:

python experiments/run_all.py --dataset-source synthetic --output experiments/results

Synthetic instances are generated by deterministic seeds and Euclidean coordinates. This provides:

1. Full reproducibility on any machine.
2. Controlled scaling by instance size.
3. Fair comparison between original and modified algorithms under identical conditions.

## Named Benchmark Mode

The project also supports named benchmark mode using a manifest CSV and JSON instance files:

python experiments/run_all.py --dataset-source named --benchmark-manifest experiments/data/instances.csv --output experiments/results

Manifest schema:

- dataset
- instance_id
- file

The loader validates file existence and instance schema before experiments start.

## Current Scope of Included Named Files

At present, the manifest includes representative examples from each benchmark family:

1. Chao benchmark family (representative sample file).
2. Dang benchmark family (representative sample file).
3. Vansteenwegen OP/TOP family (representative sample file).

This repository currently does not include all official instances from all families.
