# Results

This folder is generated by the single master script:

- `experiments/run_all.py`

## Contents

### Tables (`experiments/results/tables`)

1. `branch_and_price_results.csv`
	- Contains original and modified Branch-and-Price metrics.

2. `ils_results.csv`
	- Contains original and modified ILS metrics.

3. `presentation_summary.csv`
	- Aggregated means (original vs modified) for quick reporting in slides.

4. `bp_convergence_basic.csv`, `bp_convergence_modified.csv`
	- LP objective evolution during column generation.

5. `ils_convergence_basic.csv`, `ils_convergence_modified.csv`
	- Best-so-far curve over iterations.

### Figures (`experiments/results/figures`)

1. `bp_profit_by_n.png`, `bp_runtime_by_n.png`
2. `ils_profit_by_n.png`, `ils_runtime_by_n.png`
3. `bp_convergence_basic.png`, `bp_convergence_modified.png`
4. `ils_best_basic.png`, `ils_best_modified.png`

These are the exact figures intended for insertion into presentation slides.

## Reproduction Command

Run from repository root:

```bash
python experiments/run_all.py --dataset-source synthetic --output experiments/results
```
